# Smart Farmer Assistant (AgroAI)

An AI-Based Smart Irrigation and Crop Protection System designed to help farmers save water, protect crops, and make better farming decisions using real-time data and automation.

## Project Overview

Smart Farmer Assistant is a science and IoT-based project that monitors field conditions, automates irrigation, protects crops from animals, and provides AI-based guidance to farmers.

The system collects real-time data from the field and displays it on a dashboard while also sending alerts to the farmer via SMS.

---

## Features

- Real-time monitoring of:
  - Soil Moisture
  - Temperature
  - Humidity
- Automatic irrigation control
- SMS alerts on motor ON/OFF and animal detection
- AI chatbot for farmer guidance
- Crop protection using fencing and buzzer
- Smart scarecrow with motion detection
- Data logging for water usage analysis

---

## Technologies Used

- Microcontroller (Arduino / ESP32 / ESP8266)
- Soil Moisture Sensor
- Temperature & Humidity Sensor
- GSM Module (SMS alerts)
- Motion Sensor
- Buzzer & Motor Relay
- AI Chatbot
- Web Dashboard
